### Versoning

- Versions can be defined through branch name on github.
  - Deafult branch is always the latest version of the package.
  - In each branch, version in `carbon.json` should be corresponding to branch name.

### Publishing

1. Go to https://carbon.beanstech.tech
2. Login or create a new account.
3. Click `Add package` on navbar.
4. Fill name, description and url to package on github.
5. Click `Add package`. It's all!-

- **If github repo was used without permission your package can be deleted**
